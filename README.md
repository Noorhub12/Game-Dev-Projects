# Game Development Projects (Unity)

This repository contains my mini projects developed for the course **CSE-411: Introduction to Game Development** at  
**University of Engineering and Technology (UET), Peshawar**.

The projects demonstrate practical implementation of core Unity concepts including:
- 3D environment design
- Player movement systems
- AI behavior (basic & NavMesh-based)
- UI systems and scene management
- Physics and collision handling
- Mini-map and audio integration

---

## Projects Included

### Mini Project 1: Crescent Isle
A 3D exploration and survival-style game where the player collects resources while avoiding an AI-controlled enemy.

Folder: `Mini-Project-1_Crescent-Isle`

---

### Mini Project 2: Operation Extraction
A third-person stealth game focused on patrol-based enemy AI, animation control, and mission-based gameplay.

Folder: `Mini-Project-2_Operation-Extraction`
