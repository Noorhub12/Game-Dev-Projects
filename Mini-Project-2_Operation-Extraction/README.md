Operation Extraction is a third-person stealth game where the player infiltrates a secure facility, retrieves a data briefcase, and escapes via helipad while avoiding enemy guards.

---

## Objectives
- Implement stealth-based gameplay
- Use NavMesh for intelligent enemy movement
- Control character animations without blend trees
- Manage scenes, UI, and game states
- Design win/loss logic based on player behavior

---

## Gameplay Overview
- Avoid guards patrolling predefined paths
- Guards chase the player upon detection
- Retrieve the briefcase
- Reach the extraction helipad to win

---

## Technologies Used
- Unity 6.0
- C#
- NavMeshAgent
- Mixamo animations
- ProBuilder
- Unity UI System
