# Crescent Isle

**Mini Project 1 – CSE-411L Intro to Game Development (Spring 2025)**

Crescent Isle is a 3D exploration game developed using Unity. The player explores an island, collects resources, and avoids a hostile AI enemy called the Hunter.

---

## Objectives
- Design a terrain-based island environment
- Implement physics-based player movement
- Create collectible resources with a spawner system
- Develop basic enemy AI behavior
- Implement win and loss conditions
- Integrate UI, minimap, and environmental audio

---

## Gameplay Overview
- Collect **10 resources** to win
- Avoid the Hunter enemy
- Game freezes on win or loss
- Includes minimap and ambient water sound

---

## Technologies Used
- Unity Game Engine
- C#
- Rigidbody-based physics
- Unity Terrain System
- Unity UI
- Prefabs and collision triggers
